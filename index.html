Python 语法之缩进和注释

Python 不像C/C++那样用 { } 区分程序块，Python 用4个空格和”:” 号来区分。

打印结果：

请输入一个数字:-22
22

以 # 开始的是单行注释，多行注释有两种：

  1. 以’’’ 注释内容 ‘’’
  2. 以””” 注释内容 “””

两种都可以使用。

特别注意：

在 Python 编程中特别要注意代码的缩进，缩进错误代码是无法运行的。有了强制缩进我们写出的代码风格就更加统一漂亮了。

认识C enum 枚举

枚举类型 enum 是用来定义一组整型常量的模板。枚举量默认从零开始，我们也可以改变枚举两从指定的地方开始。

定义枚举

enum Day {Mon,Tues,Wed,Thurs,Fri,Sat,Sun}

这样我们就定义了一个枚举类型 Day; 定义之后我们就可以使用 Day 来定义枚举对象了，如下：

enum Day a;
a = Tues; // a 这时等于1
