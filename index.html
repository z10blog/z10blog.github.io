Ubuntu kivy安装

这里我们参考的是Kivy官网的安装方法，网址：https://kivy.org/#download

安装稳定版本的Kivy

  1. $ sudo add-apt-repository ppa:kivy-team/kivy
  2. $ sudo apt-get update
  3. $ sudo apt-get install python3-kivy

第三步我们安装的是python3的 kivy
如果是python2的那么要换成
$ sudo apt-get install python-kivy

安装Kivy的examples（例子）

$ sudo apt-get install python-kivy-examples

Python 函数认识

每种编程语言中都有函数的功能，函数的作用是什么？

  1. 把程序分成不同的功能块，每个函数代表了一个功能
  2. 封装逻辑实现，我们调用设计好的函数只需要输入相关的参数即可得到我们需要的结果，从而实现程序的重复使用。

要点：

  1. 默认参数必须指向不可变对象，否则程序运行时会有逻辑错误。
  2. 定义可变形参要在参数名前加’*’
  3. 调用可变参数的函数时可以直接输入多个参数，或者直接传入一个tuple，如果直接传入一个含有多个对象的tuple时需要在对象前加’*’
  4. 命名关键字参数：
    1. 命名关键字参数需要一个特殊的分隔符,后面的参数被视为命名关键字参数。
    2. 如：def person(name, age, , city, job): city和job在后边，所以就是命名关键字参数。
  5. 如果一个函数中已经有了一个可变参数，后面跟着的命名关键字参数就不需要分隔符*了
  6. 命名关键字参数必须传入参数名
  7. 命名关键字参数可以有默认值，如果有默认值则可以省略，使用默认值。
  8. Python中的函数可以混合各种参数，但是请注意，参数定义的顺序必须是：必选参数，默认参数，可变参数，命名关键字参数和关键字参数。
  9. *args 是可变参数，args接受的是一个tuple
  10. **kw 是关键字参数，kw接受的是一个dict

Python 数据类型和变量

编程就是为了处理各种数据，当然每种编程语言中都有多种类型的数据。每种编程语言都自带的内置数据类型都无法满足我们的需要，但是每种编程语言都提供了让我们创建自己的数据类型的机制。刚学首先要掌握的就是最基本的数据类型，慢慢的接触更多的和自己创建新的数据类型。

type() 认识

type() 是一个获取对象类型的函数，当我们查看某个对象的类型时可以使用。
如想获取 a 的类型我们可以这样使用：
type(a)

  1. 如果a 是一个整数则返回
  2. 如果a 是一个浮点数则返回
  3. 如果a 是一个字符串则返回
  4. 如果a 是一个函数则返回
  5. 如果a 是一个元组则返回
  6. 如果a 是一个字典则返回
  7. 如果a 是一个模块则返回
    … 等等

整数的表示方法

首先我们可以按进制区分：

1. 10进制
2. 2进制
3. 16进制 （0x8af34f 以0x开头)
4. 8进制

浮点数

浮点数就是通常说的带小数点的数字。
如：23.2，0.12，12e3,12e-4
12e3 = 1200.0
12e-4 = 0.0012
带e的表示方法是科学记数法

字符串

Python 中的字符串可以用’’、””、’’’ ‘’’、””” “””四种表示方法，最后两种效果相同。
当我们要在字符串中使用引号时需要进行转义，如下：
‘I\’m \”OK\”!’

\’ \” 代表转义成’和”
还有\n 代表换行

‘’’ ‘’’ 三个引号的字符串

‘’’hello world
I’m “OK”
end
‘’’

将会打印出引号里边的格式，3个引号内部的所有内容将会原封不动的输出。
在字符串前加上r 代表不进行转义，如遇到\n将不会换行，而是最为字符串输出。

布尔值

布尔值只有True(真)和False(假),在Python中我们可以直接使用True 和 False代表真和假。要注意大小写。

布尔值可以使用 and,or,not进行运算：
and 是与的关系只有参与运算的所有布尔值都为真时结果才为真。
or 只要参与运算的所有布尔值中有一个为真则结果就为真。
not 取反

空值 None

空值是Python里一个特殊的值，用None表示。None不代表任何数据，就是空。

变量

变量很好理解，就是可以变化的量，与之对应的是常量，常量是不能改变的量。

a = ‘hello’
b = a
a = ‘world’

  1. Python首先在内存中创建了一个字符串’hello’,然后在内存中创建了一个名为a的变量，并且指向字符串’hello’。
  2. Python在内存中创建变量b，并且把b也指向’hello’
  3. Python在内存中创建字符串’world’,并且把a 指向它。
    这时b = ‘hello’, a = ‘world’

除法

Python有几种除法：

  1. 10/3 精确除法
  2. 10 // 3 整除只保留结果的整数部分
  3. 10%3 求余 只保留余数

Python 语法之缩进和注释

Python 不像C/C++那样用 { } 区分程序块，Python 用4个空格和”:” 号来区分。

打印结果：

请输入一个数字:-22
22

以 # 开始的是单行注释，多行注释有两种：

  1. 以’’’ 注释内容 ‘’’
  2. 以””” 注释内容 “””

两种都可以使用。

特别注意：

在 Python 编程中特别要注意代码的缩进，缩进错误代码是无法运行的。有了强制缩进我们写出的代码风格就更加统一漂亮了。

认识C enum 枚举

枚举类型 enum 是用来定义一组整型常量的模板。枚举量默认从零开始，我们也可以改变枚举两从指定的地方开始。

定义枚举

enum Day {Mon,Tues,Wed,Thurs,Fri,Sat,Sun}

这样我们就定义了一个枚举类型 Day; 定义之后我们就可以使用 Day 来定义枚举对象了，如下：

enum Day a;
a = Tues; // a 这时等于1
