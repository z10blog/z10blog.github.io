Python 获取对象的信息

在编程中我们经常会要判断一个对象的类型等信息，包括对象的类型，类型有哪些方法和属性等。

在Python中使用type() 判断

type() 函数返回的是对象的类型，比如一个对象是int还是float。

types模块

如果我们要判断一个对象是否是函数我们可以使用types模块中定义的常量

types.FunctionType #函数
types.BuiltinFunctionType #内置函数
types.LambdaType #Lambda
types.GeneratoryType #生成器
